Academic Reflection

What I Learned

💡
Audience Customization
Both artifacts were designed for an academic audience — specifically, instructors and peers in a graduate AI/ML program. RecycleBot SF demonstrates applied conversational AI with domain-specific knowledge, targeting a real public need. Clean, Clear & Safe demonstrates technical depth through compound toxicity scoring and live API integration. The design choices — professional UI, technical vocabulary, real data — reflect the academic standard of rigor expected at the graduate level while remaining accessible to non-technical reviewers.
🔧
Technical Lessons
Deploying AI applications to production revealed the gap between prototype and production: API authentication, environment variables, CORS, serverless function timeouts, and JSON parsing edge cases all required careful handling. Each failure was a precise diagnostic exercise in systems thinking.
🔄
Iteration Process
The CCS Personal Shopper went through 9+ design iterations — from static mockups to a fully deployed, API-connected application. Each round of feedback sharpened both the UX and the backend architecture.
🎯
AI/ML Application
This course deepened my understanding of how foundational ML concepts — classification, scoring systems, structured inference — apply directly to production AI products. The CCS scoring algorithm is essentially a domain-specific classification model implemented via LLM prompting.
📈
Value Proposition
Both artifacts demonstrate a core MCP principle: AI is most powerful when connected to real, live data. A chatbot with static knowledge is useful. A chatbot that queries live APIs, scores results, and returns actionable recommendations is transformative.
