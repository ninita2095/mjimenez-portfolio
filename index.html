Artifact 01 — Conversational AI

RecycleBot SF

An AI-powered chatbot trained on SFPUC recycling guidelines that helps San Francisco residents instantly know where any item goes — compost, recycling, or landfill — through natural conversation.

Built using Chatbase RAG architecture, the bot retrieves and grounds its answers in official city documentation, reducing hallucination and ensuring compliance with SF's recycling rules.

Try RecycleBot SF →
Key Features
  • →Natural language recycling decisions: compost, recycle, or landfill for any item
  • →Trained on official SFPUC guidelines using Retrieval-Augmented Generation (RAG)
  • →Multi-turn conversation — handles follow-up questions and item clarifications
  • →Reduces the cognitive load of reading complex city waste guidelines
  • →Deployed as embeddable chatbot widget for public access
  • →Demonstrates domain-specific knowledge grounding for public service AI
AI/ML Concepts Demonstrated
  • →RAG (Retrieval-Augmented Generation) — grounding LLM responses in authoritative source documents
  • →Classification via prompting — mapping any item to one of three disposal categories
  • →Knowledge curation — selecting, chunking, and indexing domain documents for retrieval
  • →Conversational context management — maintaining coherent multi-turn dialogue

Artifact 02 — Real-time Data Intelligence

Clean, Clear & Safe

An AI-powered product analyzer that identifies toxic or harmful compounds in consumer products and recommends six cleaner alternatives — with live Amazon pricing and availability pulled in real time.

Combines Claude API for toxicity scoring and structured reasoning with the Amazon Data API to surface actionable, shoppable alternatives in a single interface.

Try Personal Shopper →
Key Features
  • →Paste any product — AI extracts and scores ingredient toxicity on a structured scale
  • →Generates 6 cleaner alternative products with safety rationale for each
  • →Live Amazon pricing and availability via real-time API integration
  • →Accordion UI with compound-level toxicity breakdown per ingredient
  • →Serverless architecture on Vercel — sub-second cold starts
  • →9+ design iterations from mockup to production-grade deployed app
AI/ML Concepts Demonstrated
  • →Structured inference — prompting Claude to return typed JSON with scored compound data
  • →Domain classification — toxicity scoring as an LLM-powered classification model
  • →API orchestration — chaining LLM output into a live e-commerce data pipeline
  • →Production engineering — CORS, env variables, serverless timeouts, JSON edge cases

Artifact 03 — Multi-Persona AI Coaching

The Oracle of Leaders

An AI-powered leadership coaching tool that delivers three expert perspectives simultaneously — a philosopher, a psychologist, and an iconic historical leader — each responding to the user's real leadership challenge.

Built entirely in English and Spanish, with a documented real-world case study and a concrete weekly action plan — making AI coaching accessible and actionable across languages.

Try The Oracle →
Key Features
  • →Three simultaneous expert personas: philosopher, psychologist, iconic leader
  • →Fully bilingual — responds in both English and Spanish natively
  • →Includes a documented real-world leadership case matching the user's challenge
  • →Generates a concrete, personalized weekly action plan for the user
  • →Deployed on Vercel with Claude API powering all persona responses
  • →Designed for graduate-level leadership development and executive coaching contexts
AI/ML Concepts Demonstrated
  • →Multi-persona prompt engineering — a single prompt orchestrating three distinct expert voices
  • →Multilingual generation — conditioning output language without translation post-processing
  • →Structured creative output — persona + case + action plan as a typed, predictable response schema
  • →Contextual reasoning — adapting expert perspectives to the specific leadership challenge provided
